# JuegoGato-Java-POO
Este proyecto consiste en la implementación del clásico juego del gato desarrollado en lenguaje Java, aplicando principios de programación orientada a objetos.
El sistema permite la interacción entre dos jugadores desde consola, gestionando turnos, validación de movimientos y determinación automática de un ganador o empate. Además, integra funcionalidades adicionales como el registro de resultados mediante un historial de partidas y la asignación de recompensas simbólicas a los jugadores a través de un administrador.
